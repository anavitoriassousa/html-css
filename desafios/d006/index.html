Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual TAG deverá utilizar em cada caso.

  • Título (barra de navegação)
  • Imagens
  • Hierarquia de títulos
  • Links
  • Parágrafos
  • Listas ordenadas
  • Listas não ordenadas
  • Itens
  • Citações
  • Abreviações
  • Marca texto
  • Texto menor
  • Texto deletado
  • Texto inserido
  • Texto subescrito
  • Texto sobrescrito
  • Trecho de código
  • Identação de código(trabalha em conjunto com a tag <code>
  • Quebra de linha
